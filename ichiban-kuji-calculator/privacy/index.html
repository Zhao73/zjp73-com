一番くじ電卓 プライバシーポリシー

1. 保存する情報

一番くじ名、価格、総口数、残り口数、各賞の残数、推定価値、ドロー履歴、広告非表示の購入状態など、アプリの動作に必要な情報を端末内に保存します。アカウント登録は不要です。

2. 外部サービス / SDK

  • StoreKit（アプリ内課金）
  • Google Mobile Ads（広告表示）
  • Google User Messaging Platform（広告同意管理）
  • App Tracking Transparency（広告の追跡許可確認）

3. 広告と追跡許可

本アプリは広告を表示する場合があります。広告の同意状態と ATT の許可状態に応じて、パーソナライズ広告または非パーソナライズ広告を利用します。設定は iPhone / iPad の設定アプリおよびアプリ内の広告プライバシー設定から見直せます。

4. ユーザーができること

  • アプリ設定から端末内データを削除する
  • 広告のプライバシー設定を見直す
  • 追跡許可をシステム設定から変更する

5. お問い合わせ

本ポリシーに関するお問い合わせは、support@zjp73.com までお願いします。
